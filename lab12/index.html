Challenges

In this lab, the challenges provided was to include dividers with special CSS properties that includes centering and floating. We were tasked in making four dividers based on testing these new functions as one would be tested the floating property with text and three other diveders are tested on centering elements such as text and images.

Problems

So, I eliminated some of the unnecessary CSS code that was irrelevant to this lab's focus and reorganized the HTML to get a better view of my code. Then, I started experimenting with floating and centering properties in the CSS. The centering properties were more difficult than understanding floating as some text and image properties wouldn't stick to the middle of the dividers.
